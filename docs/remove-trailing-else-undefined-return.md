### `remove-trailing-else-undefined-return`

Removes useless returns in trailing else statements generated by Decaf.

```sh
jscodeshift -t shopify-codemod/transforms/remove-trailing-else-undefined-return <file>
```

#### Example

```js
foo = () => {
  if (bar != null) {
    return bar(true);
  } else {
    console.log('foo');
    return void 0;
  }
}

// BECOMES:

foo = () => {
  if (bar != null) {
    return bar(true);
  } else {
    console.log('foo');
  }
}
```

```js
foo = () => {
  if (bar != null) {
    return bar(true);
  } else {
    return void 0;
  }
}

// BECOMES:

foo = () => {
  if (bar != null) {
    return bar(true);
  }
}
```
